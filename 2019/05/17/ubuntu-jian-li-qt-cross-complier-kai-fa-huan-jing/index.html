  1. toolchain (arm 開發板廠商提供 or “sudo apt-get install gcc-arm-linux-gnueabi”)

  3. qt-everywhere-opensource-src-4.8.6.tar.gz
  4. tslib-1.4.tar.gz
  • qt-creator
  • qt-everywhere-opensource

$wget
$sudo chmod +x qt-opensource-linux-x64-5.9.7.run
