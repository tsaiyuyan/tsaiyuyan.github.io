Ubuntu加入新的帳戶
ubuntu
Linux
發布日期:   2018-02-13
文章字數:   218
閱讀時長:   1 分
閱讀次數:  

原本目的:以前開 AWS 會拿到一組金鑰，但每次都用鑰匙 ssh 登入連線實在很不方便。

1.Ubuntu 加入新的帳戶 (tsaiyuyan 為例)

$sudo adduser tsaiyuyan

2.(可選) 可加入 sudo group

$sudo adduser <username> sudo    ===><username>  這邊示範是 tsaiyuyan

3.開始設定屬性

$sudo visudo    會有一個檔案
tsaiyuyan ALL=(ALL) ALL   加入此參數
ctrl +x 離開 ->問說是否Save ->選擇yes ->會跑出一些啥其他的命令，直接按Enter就可以了不用管

$sudo vim /etc/ssh/sshd_config

(PasswordAuthentication)打開以後，找到這一行！
將它改成yes!
接著儲存離開！

然後就重開 ssh，指令如下：

$sudo /etc/init.d/ssh restart

完成以後，你就能使用帳號、密碼登入了！
不用每次都要附上 Key 這麼麻煩！！！
如果還是不能登入，就使用重新開機在嘗試。

$shutdown -r now
